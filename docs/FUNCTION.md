# 機能要件

## 1. 対象コンテンツ

収集一覧と、以下の収集対象ごとの個別ページを提供します。

- 野鳥
- 魚
- 昆虫
- グルメ

## 2. 共通ナビゲーション

- 収集一覧と各収集ページの間を移動できること。
- 現在表示中のページを識別できること。
- 収集一覧に、各収集ページへのリンクと登録件数を表示すること。

## 3. 一覧表示

### 野鳥・魚・昆虫

各項目に以下の情報を表示します。該当しない情報は表示しません。

- 画像
- 名前
- 分類
- 解放レベル
- イベント名
- 出現天気
- 出現時間
- 出現場所
- 星1売値
- 星評価
- 解放条件や注記などの補足情報

種別固有の情報は以下です。

- 野鳥：撮影距離、星5撮影条件の天気・時間・補足
- 魚：サイズ
- 昆虫：出現方法

### グルメ

各グルメに以下の情報を表示します。該当しない情報は表示しません。

- グルメ画像
- グルメ名
- 分類
- 解放レベル
- イベント名
- 材料の画像と名称
- 星1売値
- 星評価

材料画像が存在しない場合も、材料名は表示します。

## 4. 検索・絞り込み

- 野鳥・魚・昆虫は、名前、出現条件、種別固有情報、補足情報、売値を検索対象とすること。
- グルメは、グルメ名、材料名、イベント名を検索対象とすること。
- 複数の検索語を指定した場合は、すべての語に一致する項目を表示すること。

利用できる絞り込み条件は以下です。

| 対象 | 分類 | その他の条件 |
| --- | --- | --- |
| 野鳥 | 通常、上級、イベント | 天気、出現場所 |
| 魚 | 川、湖、海、上級、イベント | 天気、出現場所 |
| 昆虫 | 通常、上級、イベント | 天気、出現場所 |
| グルメ | 通常、上級、イベント | なし |

全収集ページで、星1〜4評価済み項目と星5評価済み項目をそれぞれ非表示にできます。未評価項目は星1〜4評価済み項目に含めません。

検索または絞り込みの結果として、表示件数を更新します。該当項目が存在しない場合は、その旨と条件を初期化する操作を表示します。

## 5. 並び替え

野鳥・魚・昆虫では以下の並び順を選択できます。

- 掲載順
- 解放レベル順
- 名前順
- 星5未達成項目を優先

グルメでは以下の並び順を選択できます。

- 掲載順
- 解放レベル順
- 名前順
- 星1売値が高い順
- 星5未達成項目を優先

## 6. 星評価と達成率

- 各項目に星1〜5の評価を設定できること。
- 評価をブラウザのローカルストレージへ保存すること。
- 保存した評価を収集対象および項目ごとに識別し、ページを再読込しても維持すること。
- 評価変更を一覧、絞り込み、並び順、星5達成率へ即時反映すること。
- 各収集ページに、星5評価済み件数、全件数、達成率を表示すること。

## 7. 売値

- 星1時点の売値を表示すること。
- 通常売値とイベント売値を扱い、通貨種別を識別できること。
- 2種類の売値が存在する場合は、それぞれを別の行に表示すること。

## 8. 画像の欠損・読込失敗

項目画像または材料画像が存在しない場合や読込に失敗した場合も、一覧表示を継続し、名称などの情報を確認できること。
